Style Guide

Reference guide for UI consistency, accessibility, and aesthetic decisions across the project.

Colour Scheme

A dark-first palette with key UI highlights. Prioritizes contrast, readability, and visual calmness.

Primary

#00B3FF

Used for buttons, highlights, and links.

Accent

#F18F01

Used for hover effects and subtle CTAs.

Surface

#1E1E1E

Main background surface. Low strain for dark UIs.

Text

#F0F0F0

Used for all body and heading text. High contrast.

Typography

Consistent type sizes with hierarchy. All text uses Nunito.

Heading XL

Font: Nunito, 48px, Extra Bold

Used for hero titles, banners, and CTAs.

Heading Large

Font: Nunito, 36px, Bold

Used for section titles and key prompts.

Heading Medium

Font: Nunito, 28px, Semi-Bold

Used for sub-sections and component titles.

Body Large

Font: Nunito, 20px, Regular

Used for descriptions and intro text.

Body Medium

Font: Nunito, 16px, Regular

Default paragraph style.

Caption

Font: Nunito, 12px, Medium

For metadata and low-emphasis info.

Buttons

Accessible, responsive CTA buttons.

Link Styles

Meets WCAG 2.1. Accessible via keyboard navigation.

Default Link | Hover Link | Focus Link | Disabled Link

Each state (normal, hover, focus, disabled) supports accessibility expectations and usability.
