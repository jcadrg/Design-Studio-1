Project Structure

This site provides an overview of the project’s file organization to help developers and designers quickly understand the modular layout.

📁 /

  • index.html — Main entry point of the application
  • landing-page.html — The hero homepage with parallax
  • more-details.html — Fish profile extended detail view
  • style-guide.html — UI style and brand guidelines

📁 css/

  • style.css — Shared base styles
  • landing.css — Landing page specific styles
  • details.css — Species detail view styles
  • style-guide.css — Styles for the guide page
  • style-structure.css — Styles for this structure view

📁 js/

  • chart.js — Chart rendering logic (Chart.js)
  • filters.js — Dynamic filtering for index page
  • details.js — Detail page species logic

📁 json/

  • fish_samples_cleaned.json — Main ecological dataset
  • murray_darling_basin.geojson — Map boundary data
  • species_common_names.json — Scientific to common name mapping

📁 images/

  • Hero and background images, species thumbnails
